BUSCADOR DE CEP - DOCUMENTAÇÃO

1. FUNCIONALIDADES IMPLEMENTADAS
- Busca de CEP via API ViaCEP
- Interface responsiva para mobile e desktop
- Validação de CEP (8 dígitos)

- Limpar busca
- SEO otimizado
